1. 安装samba服务
sudo apt-get install samba samba-common
2.创建一个用于分享的samba目录
一定要记住此时的目录，接下来都需要使用
sudo mkdir /home/share 
3.设置分享目录的权限
sudo chmod 777 /home/share
4.添加samba用户
添加samba用户，为了方便windows下的连接，这里建议将samba用户设置成与Ubuntu用户信息一致（用户名、密码），方便！！！
sudo smbpasswd -a xxx // xxx:用户名字
5.配置samba的配置文件
sudo gedit /etc/samba/smb.conf
在配置文件末尾添加下面的文本

[share]
comment = share folder
browseable = yes
path = /home/share
create mask = 0700
directory mask = 0700
valid users = cfj
force user = cfj
force group = cfj
public = yes
available = yes
writable = yes

6.重启samba服务
sudo service smbd restart

7.在windows下运行
Win+R 在运行框内输入 \\ip 即可访问，这里的ip地址是Ubuntu的ip地址

之后会提示，需要输入samba用户名及密码（这里是我们之前设置的smaba用户）
然后就可以在Ubuntu与windows共享文件了。
